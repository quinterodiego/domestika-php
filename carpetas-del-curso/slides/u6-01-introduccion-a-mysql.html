Introducción a MySQL

¿Qué es?

Sistema de Base de Datos relaciones, gratuíto y fácil de utilizar.

Tablas

MySQL guarda todos los datos en tablas. Una tabla es una colección de datos relacionados con la misma estructura.

+----+------------+-----------+----------------------+
| id | first_name | last_name | email                |
+----+------------+-----------+----------------------+
|  1 | Peter      | Parker    | peterparker@mail.com |
|  2 | John       | Rambo     | johnrambo@mail.com   |
|  3 | Clark      | Kent      | clarkkent@mail.com   |
|  4 | John       | Carter    | johncarter@mail.com  |
|  5 | Harry      | Potter    | harrypotter@mail.com |
+----+------------+-----------+----------------------+

SQL, (Structured Query Language)

El lenguaje que utiliza MySQL para sacar, insertar, actualizar o borrar datos en una base de datos.


						SELECT * FROM persons WHERE first_name = 'Clark';
					


						INSERT INTO persons
						( first_name, last_name, email )
						VALUES
						( 'Bruce', 'Wayne', 'brucewayne@mail.com' );
					


						DELETE FROM persons WHERE id = 4;
					

						UPDATE persons SET email = 'batman@mail.com'
						WHERE first_name = 'Bruce'
						AND last_name = 'Wayne';
