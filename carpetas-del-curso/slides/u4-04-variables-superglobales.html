$_SERVER

Array que contiene información como cabeceras, rutas y ubicaciones de script

Depende del servidor, algunos pueden omitir algunos campos


					  HTTP_HOST - 'localhost:8888'
					  HTTP_USER_AGENT - 'Mozilla/5.0 (Macintosh; Intel Mac OS X 10_13_5) AppleWebKit/537.36 (KHTML, like Gecko) Chrome/70.0.3538.110 Safari/537.36'
					  HTTP_REFERER - 'http://localhost:8888/'
					  SERVER_NAME - 'localhost'
					  SERVER_PORT - '8888'
					  DOCUMENT_ROOT - '/Applications/MAMP/htdocs'
					  SCRIPT_FILENAME - '/Applications/MAMP/htdocs/tu-proyecto/index.php'
					  REQUEST_METHOD - 'GET'
					  REQUEST_URI - '/tu-proyecto/'

$_GET

Array que contiene los datos que pasan por la URL


						miurl.com?nombre=Ignacio&apellido1=Cruz&apellido2=Moreno
					

						echo $_GET['nombre']; // Ignacio
					

						echo $_GET['apellido1']; // Cruz
					

						echo $_GET['apellido2']; // Moreno

$_POST

Array que contiene los datos que pasan a través de un request de tipo POST (Normalmente un formulario)


						echo $_POST['nombre']; // Ignacio
					

						echo $_POST['apellido1']; // Cruz
					

						echo $_POST['apellido2']; // Moreno

Otras superglobales

  • $_FILES: Array asociativo de archivos subidos a través de un formulario con método POST
  • $_COOKIE: Array de Cookies del usuario que visita el sitio.
  • $_SESSION: Array donde se puede guardar información sobre un usuario. Al recargar, dicha información no se pierde.
  • $_REQUEST: Contiene $_GET, $POST y $_COOKIE.
