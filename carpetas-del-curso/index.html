Curso de Introducción a PHP

La carpeta del curso está dividida en:

  • ./slides: Contiene las slides utilizadas durante el curso aunque es mejor acceder a ella desde el menú Lecciones
  • ./carpeta-proyecto: Aquí está el código de todo el proyecto dividido en unidades y Proyecto Final.

    Dentro de cada unidad hay varias carpetas dependiendo del número de lección y dentro de cada lección hay dos carpetas: 01-inicio y 02-final que suponen el código con el que se empieza y se acaba la lección.

    Puedes copiar cualquiera de esas dos carpetas a la carpeta de tu proyecto dentro de htdocs en MAMP para practicar una lección en concreto.

  • ./tu-proyecto Es un simple archivo php. Puedes copiar esta carpeta a tu htdocs y empezar el curso desde ahí.
  • ./assets Imágenes y CSS para esta página (no tiene nada que ver con el proyecto).
